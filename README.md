# Solv Team Skills

A collection of personal skills pages for the Solv team.

## Structure

Each team member has their own folder:

```
/your-name/index.html
```

## How to add your skills

1. Fork or clone this repo
2. Create a folder with your name (lowercase, no spaces)
3. Add your `index.html` inside it
4. Open a PR — or just push directly if you have access
